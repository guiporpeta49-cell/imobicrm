Ajustes responsivos aplicados nas páginas principais.

O pacote inclui versões ajustadas de:
- Dashboard
- Clientes
- Corretores
- Imóveis
- Imóvel Detalhe
- Visitas
- Negociações
- Login
- Super Admin

Principais melhorias:
- ações do cabeçalho empilhando no celular
- campos de busca ocupando largura total no mobile
- botões empilhando no mobile
- cards e áreas de detalhe com alturas melhores para telas pequenas
- rodapé fixo do modal de imóveis
